## ï¿½ Usage

1. **Input**: Paste the message or post you want to reply to into the **Source Input** field.
2. **Customize**: 
   - Select a **Tone** that matches your desired vibe.
   - Choose the preferred **Length**.
   - (Optional) Add **Custom Refinement** instructions (e.g., "Ask them to join my free workshop").
3. **Generate**: Click the **Generate** button.
4. **Polish**: Use the **Refine** dropdown to tweak the output (e.g., "Add relevant emojis").
5. **Copy**: Click **Copy** to save the response to your clipboard and paste it into Skool!

> **Pro Tip**: Enable **Auto-Copy** in the settings to skip the manual copy step!
